# Virtual DOM

–í–æ–∑–º–æ–∂–Ω–æ, –≤—ã —Å–ª—ã—à–∞–ª–∏, –∫–∞–∫ –ª—é–¥–∏ —É–ø–æ–º–∏–Ω–∞—é—Ç ¬´Virtual DOM¬ª, –∏ –∑–∞–¥–∞–≤–∞–ª–∏—Å—å –≤–æ–ø—Ä–æ—Å–æ–º: —á—Ç–æ –¥–µ–ª–∞–µ—Ç –µ–≥–æ ¬´–≤–∏—Ä—Ç—É–∞–ª—å–Ω—ã–º¬ª? –ß–µ–º ¬´–≤–∏—Ä—Ç—É–∞–ª—å–Ω—ã–π¬ª DOM –æ—Ç–ª–∏—á–∞–µ—Ç—Å—è –æ—Ç —Ä–µ–∞–ª—å–Ω–æ–≥–æ DOM, –∫–æ—Ç–æ—Ä—ã–π –º—ã –∏—Å–ø–æ–ª—å–∑—É–µ–º –ø—Ä–∏ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–∏ –¥–ª—è –±—Ä–∞—É–∑–µ—Ä–∞?

[Virtual DOM](https://doka.guide/tools/react-and-alternatives/#virtual-dom) ‚Äî —ç—Ç–æ –ø—Ä–æ—Å—Ç–æ–µ –æ–ø–∏—Å–∞–Ω–∏–µ –¥—Ä–µ–≤–æ–≤–∏–¥–Ω–æ–π —Å—Ç—Ä—É–∫—Ç—É—Ä—ã —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –æ–±—ä–µ–∫—Ç–æ–≤:

```js
let vdom = {
  type: 'p', // —ç–ª–µ–º–µ–Ω—Ç <p>
  props: {
    class: 'big', // —Å –∫–ª–∞—Å—Å–æ–º "big"
    children: [
      '–ü—Ä–∏–≤–µ—Ç, –º–∏—Ä!', // –∏ —Å —Ç–µ–∫—Å—Ç–æ–º "–ü—Ä–∏–≤–µ—Ç, –º–∏—Ä!"
    ],
  },

```

–¢–∞–∫–∏–µ –±–∏–±–ª–∏–æ—Ç–µ–∫–∏, –∫–∞–∫ Preact, –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è—é—Ç –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å —Å–æ–∑–¥–∞–≤–∞—Ç—å –æ–ø–∏—Å–∞–Ω–∏—è, –∫–æ—Ç–æ—Ä—ã–µ –∑–∞—Ç–µ–º —Å—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è —Å –¥–µ—Ä–µ–≤–æ–º DOM –±—Ä–∞—É–∑–µ—Ä–∞. –í –∏—Ç–æ–≥–µ —ç—Ç–æ –¥–µ—Ä–µ–≤–æ –æ–±–Ω–æ–≤–ª—è–µ—Ç—Å—è –≤ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏–∏ —Å –æ–ø–∏—Å–∞–Ω–Ω–æ–π —Å—Ç—Ä—É–∫—Ç—É—Ä–æ–π –≤–∏—Ä—Ç—É–∞–ª—å–Ω–æ–≥–æ –¥–µ—Ä–µ–≤–∞.

–≠—Ç–æ –ø–æ–ª–µ–∑–Ω—ã–π –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç, –ø–æ—Å–∫–æ–ª—å–∫—É –æ–Ω –ø–æ–∑–≤–æ–ª—è–µ—Ç –Ω–∞–º —Å–æ—Å—Ç–∞–≤–ª—è—Ç—å –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–µ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å—ã _–¥–µ–∫–ª–∞—Ä–∞—Ç–∏–≤–Ω–æ_, –∞ –Ω–µ _–∏–º–ø–µ—Ä–∞—Ç–∏–≤–Ω–æ_. –í–º–µ—Å—Ç–æ —Ç–æ–≥–æ —á—Ç–æ–±—ã –æ–ø–∏—Å—ã–≤–∞—Ç—å _–∫–∞–∫_ –æ–±–Ω–æ–≤–ª—è—Ç—å DOM –≤ —Ä–µ–∞–∫—Ü–∏—é –Ω–∞ —Ç–∞–∫–∏–µ –≤–µ—â–∏, –∫–∞–∫ –≤–≤–æ–¥ —Å –∫–ª–∞–≤–∏–∞—Ç—É—Ä—ã –∏–ª–∏ –º—ã—à–∏, –Ω–∞–º –Ω—É–∂–Ω–æ —Ç–æ–ª—å–∫–æ –æ–ø–∏—Å–∞—Ç—å _–∫–∞–∫_ DOM –¥–æ–ª–∂–µ–Ω –≤—ã–≥–ª—è–¥–µ—Ç—å –ø–æ—Å–ª–µ –ø–æ–ª—É—á–µ–Ω–∏—è –≤—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –º—ã –º–æ–∂–µ–º –Ω–µ–æ–¥–Ω–æ–∫—Ä–∞—Ç–Ω–æ –¥–∞–≤–∞—Ç—å Preact –æ–ø–∏—Å–∞–Ω–∏–µ –¥—Ä–µ–≤–æ–≤–∏–¥–Ω—ã—Ö —Å—Ç—Ä—É–∫—Ç—É—Ä, –∏ –æ–Ω –±—É–¥–µ—Ç –æ–±–Ω–æ–≤–ª—è—Ç—å DOM –±—Ä–∞—É–∑–µ—Ä–∞ –¥–ª—è —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏—è –∫–∞–∂–¥–æ–º—É –Ω–æ–≤–æ–º—É –æ–ø–∏—Å–∞–Ω–∏—é ‚Äî –Ω–µ–∑–∞–≤–∏—Å–∏–º–æ –æ—Ç –µ–≥–æ —Ç–µ–∫—É—â–µ–π —Å—Ç—Ä—É–∫—Ç—É—Ä—ã.

–í —ç—Ç–æ–π –≥–ª–∞–≤–µ –º—ã —É–∑–Ω–∞–µ–º, –∫–∞–∫ —Å–æ–∑–¥–∞–≤–∞—Ç—å –¥–µ—Ä–µ–≤—å—è Virtual DOM –∏ –∫–∞–∫ —É–∫–∞–∑—ã–≤–∞—Ç—å Preact –æ–±–Ω–æ–≤–ª—è—Ç—å DOM –≤ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏–∏ —Å —ç—Ç–∏–º–∏ –¥–µ—Ä–µ–≤—å—è–º–∏.

### –°–æ–∑–¥–∞–Ω–∏–µ –¥–µ—Ä–µ–≤—å–µ–≤ Virtual DOM

–°—É—â–µ—Å—Ç–≤—É–µ—Ç –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å–ø–æ—Å–æ–±–æ–≤ —Å–æ–∑–¥–∞–Ω–∏—è —Ç–∞–∫–∏—Ö –¥–µ—Ä–µ–≤—å–µ–≤:

- `createElement()`: —Ñ—É–Ω–∫—Ü–∏—è, –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ–º–∞—è Preact
- [JSX]: HTML-–ø–æ–¥–æ–±–Ω—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å, –∫–æ—Ç–æ—Ä—ã–π –º–æ–∂–µ—Ç –±—ã—Ç—å —Å–∫–æ–º–ø–∏–ª–∏—Ä–æ–≤–∞–Ω –≤ JavaScript
- [HTM]: HTML-–ø–æ–¥–æ–±–Ω—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å, –∫–æ—Ç–æ—Ä—ã–π –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –Ω–µ–ø–æ—Å—Ä–µ–¥—Å—Ç–≤–µ–Ω–Ω–æ –≤ JavaScript

–ù–∞—á–∞—Ç—å —Å—Ç–æ–∏—Ç —Å —Å–∞–º–æ–≥–æ –ø—Ä–æ—Å—Ç–æ–≥–æ –ø–æ–¥—Ö–æ–¥–∞, –∫–æ—Ç–æ—Ä—ã–π –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ –Ω–µ–ø–æ—Å—Ä–µ–¥—Å—Ç–≤–µ–Ω–Ω–æ–º –≤—ã–∑–æ–≤–µ —Ñ—É–Ω–∫—Ü–∏–∏ Preact `createElement()`:

```jsx
import { createElement, render } from 'preact';

let vdom = createElement(
  'p', // —ç–ª–µ–º–µ–Ω—Ç <p>
  { class: 'big' }, // —Å –∫–ª–∞—Å—Å–æ–º "big"
  '–ü—Ä–∏–≤–µ—Ç, –º–∏—Ä!' // –∏ —Å —Ç–µ–∫—Å—Ç–æ–º "–ü—Ä–∏–≤–µ—Ç, –º–∏—Ä!"
);

render(vdom, document.body);
```

–ü—Ä–∏–≤–µ–¥—ë–Ω–Ω—ã–π –≤—ã—à–µ –∫–æ–¥ —Å–æ–∑–¥–∞–µ—Ç –≤ Virtual DOM ¬´–æ–ø–∏—Å–∞–Ω–∏–µ¬ª —ç–ª–µ–º–µ–Ω—Ç–∞ –∞–±–∑–∞—Ü–∞. –ü–µ—Ä–≤—ã–º –∞—Ä–≥—É–º–µ–Ω—Ç–æ–º createElement —è–≤–ª—è–µ—Ç—Å—è –∏–º—è HTML-—ç–ª–µ–º–µ–Ω—Ç–∞. –í—Ç–æ—Ä–æ–π –∞—Ä–≥—É–º–µ–Ω—Ç ‚Äî ¬´—Ä–µ–∫–≤–∏–∑–∏—Ç¬ª —ç–ª–µ–º–µ–Ω—Ç–∞ ‚Äî –æ–±—ä–µ–∫—Ç, —Å–æ–¥–µ—Ä–∂–∞—â–∏–π –∞—Ç—Ä–∏–±—É—Ç—ã (–∏–ª–∏ —Å–≤–æ–π—Å—Ç–≤–∞) –¥–ª—è —É—Å—Ç–∞–Ω–æ–≤–∫–∏ –Ω–∞ —ç–ª–µ–º–µ–Ω—Ç. –õ—é–±—ã–µ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã —è–≤–ª—è—é—Ç—Å—è –¥–æ—á–µ—Ä–Ω–∏–º–∏ –¥–ª—è —ç–ª–µ–º–µ–Ω—Ç–∞, –∏ –º–æ–≥—É—Ç –±—ã—Ç—å —Å—Ç—Ä–æ–∫–∞–º–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä `'–ü—Ä–∏–≤–µ—Ç, –º–∏—Ä!'`) –∏–ª–∏ —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏ Virtual DOM –æ—Ç –≤–ª–æ–∂–µ–Ω–Ω—ã—Ö –≤—ã–∑–æ–≤–æ–≤ `createElement()`.

–ü–æ—Å–ª–µ–¥–Ω—è—è —Å—Ç—Ä–æ–∫–∞ –¥–∏–∫—Ç—É–µ—Ç Preact –ø–æ—Å—Ç—Ä–æ–∏—Ç—å —Ä–µ–∞–ª—å–Ω–æ–µ –¥–µ—Ä–µ–≤–æ DOM, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–µ–µ ¬´–æ–ø–∏—Å–∞–Ω–∏—é¬ª Virtual DOM, –∏ –≤—Å—Ç–∞–≤–∏—Ç—å —ç—Ç–æ DOM-–¥–µ—Ä–µ–≤–æ –≤ `<body>` –≤–µ–±-—Å—Ç—Ä–∞–Ω–∏—Ü—ã.

### –î–æ–±–∞–≤–∏–º JSX!

–ú—ã –º–æ–∂–µ–º –ø–µ—Ä–µ–ø–∏—Å–∞—Ç—å –ø—Ä–µ–¥—ã–¥—É—â–∏–π –ø—Ä–∏–º–µ—Ä, –∏—Å–ø–æ–ª—å–∑—É—è [JSX], –Ω–µ –º–µ–Ω—è—è –µ–≥–æ —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç—å. JSX –ø–æ–∑–≤–æ–ª—è–µ—Ç –Ω–∞–º –æ–ø–∏—Å—ã–≤–∞—Ç—å –Ω–∞—à —ç–ª–µ–º–µ–Ω—Ç –∞–±–∑–∞—Ü–∞, –∏—Å–ø–æ–ª—å–∑—É—è —Å–∏–Ω—Ç–∞–∫—Å–∏—Å, –ø–æ–¥–æ–±–Ω—ã–π HTML, —á—Ç–æ –º–æ–∂–µ—Ç –ø–æ–º–æ—á—å —Å–æ—Ö—Ä–∞–Ω–∏—Ç—å —á–∏—Ç–∞–±–µ–ª—å–Ω–æ—Å—Ç—å –ø—Ä–∏ –æ–ø–∏—Å–∞–Ω–∏–∏ –±–æ–ª–µ–µ —Å–ª–æ–∂–Ω—ã—Ö –¥–µ—Ä–µ–≤—å–µ–≤. –ù–µ–¥–æ—Å—Ç–∞—Ç–æ–∫ JSX –≤ —Ç–æ–º, —á—Ç–æ –Ω–∞—à –∫–æ–¥ –±–æ–ª—å—à–µ –Ω–µ –ø–∏—à–µ—Ç—Å—è –Ω–∞ JavaScript –∏ –¥–æ–ª–∂–µ–Ω –∫–æ–º–ø–∏–ª–∏—Ä–æ–≤–∞—Ç—å—Å—è —Å –ø–æ–º–æ—â—å—é —Ç–∞–∫–æ–≥–æ –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç–∞, –∫–∞–∫ [Babel]. –ö–æ–º–ø–∏–ª—è—Ç–æ—Ä—ã –ø—Ä–µ–æ–±—Ä–∞–∑—É—é—Ç –ø—Ä–∏–≤–µ–¥—ë–Ω–Ω—ã–π –Ω–∏–∂–µ –ø—Ä–∏–º–µ—Ä JSX –≤ —Ç–æ—á–Ω—ã–π –∫–æ–¥ `createElement()`, –∫–æ—Ç–æ—Ä—ã–π –º—ã –≤–∏–¥–µ–ª–∏ –≤ –ø—Ä–µ–¥—ã–¥—É—â–µ–º –ø—Ä–∏–º–µ—Ä–µ.

```jsx
import { createElement, render } from 'preact';

let vdom = <p class='big'>–ü—Ä–∏–≤–µ—Ç, –º–∏—Ä!</p>;

render(vdom, document.body);
```

–¢–µ–ø–µ—Ä—å —ç—Ç–æ –≥–æ—Ä–∞–∑–¥–æ –±–æ–ª—å—à–µ –ø–æ—Ö–æ–∂–µ –Ω–∞ HTML!

–ò –ø–æ—Å–ª–µ–¥–Ω–µ–µ, —á—Ç–æ —Å–ª–µ–¥—É–µ—Ç –ø–æ–º–Ω–∏—Ç—å –æ JSX: –∫–æ–¥ –≤–Ω—É—Ç—Ä–∏ JSX-—ç–ª–µ–º–µ–Ω—Ç–∞ (–≤ —É–≥–ª–æ–≤—ã—Ö —Å–∫–æ–±–∫–∞—Ö) ‚Äî —ç—Ç–æ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å, –∞ –Ω–µ JavaScript. –î–ª—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è JavaScript —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–∞, –Ω–∞–ø—Ä–∏–º–µ—Ä, —á–∏—Å–µ–ª –∏–ª–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö, —Å–Ω–∞—á–∞–ª–∞ –Ω—É–∂–Ω–æ ¬´–ø–µ—Ä–µ–ø—Ä—ã–≥–Ω—É—Ç—å¬ª –æ–±—Ä–∞—Ç–Ω–æ –∏–∑ JSX —Å –ø–æ–º–æ—â—å—é `{–≤—ã—Ä–∞–∂–µ–Ω–∏—è}` ‚Äî –∞–Ω–∞–ª–æ–≥–∏—á–Ω–æ –ø–æ–ª—è–º –≤ —à–∞–±–ª–æ–Ω–µ. –í –ø—Ä–∏–≤–µ–¥—ë–Ω–Ω–æ–º –Ω–∏–∂–µ –ø—Ä–∏–º–µ—Ä–µ –ø–æ–∫–∞–∑–∞–Ω—ã –¥–≤–∞
–≤—ã—Ä–∞–∂–µ–Ω–∏—è: –æ–¥–Ω–æ –¥–ª—è —É—Å—Ç–∞–Ω–æ–≤–∫–∏ `class` –≤ –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω—É—é —Å—Ç—Ä–æ–∫—É, –∞ –¥—Ä—É–≥–æ–µ –¥–ª—è –≤—ã—á–∏—Å–ª–µ–Ω–∏—è —á–∏—Å–ª–∞.

```jsx

//                  ^---JS---^         ^--JS--^
```

–ï—Å–ª–∏ –º—ã –≤—ã–ø–æ–ª–Ω–∏–º –∫–æ–º–∞–Ω–¥—É `render(vdom, document.body)`, —Ç–æ –Ω–∞ —ç–∫—Ä–∞–Ω–µ –ø–æ—è–≤–∏—Ç—Å—è —Ç–µ–∫—Å—Ç ¬´–ü—Ä–∏–≤–µ—Ç, 42!¬ª.

### –ï—â—ë —Ä–∞–∑ —Å HTM

[HTM] ‚Äî —ç—Ç–æ –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–∞ JSX, –∏—Å–ø–æ–ª—å–∑—É—é—â–∞—è —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–µ —à–∞–±–ª–æ–Ω—ã —Å —Ç–µ–≥–∞–º–∏ JavaScript, —Ç–µ–º —Å–∞–º—ã–º —É—Å—Ç—Ä–∞–Ω—è—è –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç—å –≤ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–µ. –ï—Å–ª–∏ –≤—ã –µ—â—ë –Ω–µ —Å—Ç–∞–ª–∫–∏–≤–∞–ª–∏—Å—å —Å —à–∞–±–ª–æ–Ω–∞–º–∏ —Ç–µ–≥–æ–≤, –æ–Ω–∏ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç —Å–æ–±–æ–π —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–π —Ç–∏–ø –ª–∏—Ç–µ—Ä–∞–ª–∞ String, –∫–æ—Ç–æ—Ä—ã–π –º–æ–∂–µ—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å –ø–æ–ª—è `${–≤—ã—Ä–∞–∂–µ–Ω–∏–π}`:


–í HTM –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è `${–≤—ã—Ä–∞–∂–µ–Ω–∏–µ}` –≤–º–µ—Å—Ç–æ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–∞ `{–≤—ã—Ä–∞–∂–µ–Ω–∏–µ}` –∏–∑ JSX, –æ–±–ª–µ–≥—á–∞—è –ø–æ–Ω–∏–º–∞–Ω–∏–µ, –∫–∞–∫–∏–µ —á–∞—Å—Ç–∏ –≤–∞—à–µ–≥–æ –∫–æ–¥–∞ —è–≤–ª—è—é—Ç—Å—è —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏ HTM/JSX, –∞ –∫–∞–∫–∏–µ —á–∞—Å—Ç–∏ —è–≤–ª—è—é—Ç—Å—è –æ–±—ã—á–Ω—ã–º JavaScript:

```js

//                        ^---JS---^          ^--JS--^
```

–í—Å–µ —ç—Ç–∏ –ø—Ä–∏–º–µ—Ä—ã –ø—Ä–∏–≤–æ–¥—è—Ç –∫ –æ–¥–Ω–æ–º—É –∏ —Ç–æ–º—É –∂–µ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—É: –¥–µ—Ä–µ–≤–æ Virtual DOM, –∫–æ—Ç–æ—Ä–æ–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –ø–µ—Ä–µ–¥–∞–Ω–æ –≤ Preact –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –∏–ª–∏ –æ–±–Ω–æ–≤–ª–µ–Ω–∏—è —Å—É—â–µ—Å—Ç–≤—É—é—â–µ–≥–æ DOM-–¥–µ—Ä–µ–≤–∞.
